# StegEntity Role Model

StegEntity is the runtime environment for AI-role expressions in StegVerse.

- `StegBot`: bounded automation.
- `StegAgent`: reasoning, review, proposal, and classification.
- `StegEntity`: governed executable entity.
- `StegVerseAI`: StegVerse-side counterpart intelligence.
- `UserAI`: user-bound delegated intelligence.

All roles must operate through StegID identity, TV/TVC scoped authority, maintenance capsules, adapters, receipts, and verification.
